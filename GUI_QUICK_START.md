## 🎬 5分钟快速体验

### 第一步：启动 GUI

```powershell
cd d:\MyQuantProject
streamlit run GUI_Client/app_v2.py
```

浏览器会自动打开：http://localhost:8501

### 第二步：选择标的和策略

```
侧边栏：
✓ 选择标的：AAPL
✓ 开始日期：2023-01-01
✓ 结束日期：今天
✓ 初始资金：30000
```

### 第三步：选择分歧交易策略（改进版）

在"策略选择"下拉框中选择：**分歧交易策略（改进版）**

### 第四步：调整参数（新功能！）

你会看到 6 个滑块参数：

```
📊 趋势均线周期：10-50（默认20）
   ↳ 用于判断上升/下降趋势

📈 波幅扩大倍数：1.1-2.0（默认1.3）
   ↳ B日波幅需超过A日波幅的倍数
   >>> 更高 = 更严格，交易更少但质量更好

📢 成交量倍数：1.0-2.0（默认1.2）
   ↳ 成交量需超过20日均量的倍数

🎯 ATR周期：7-30（默认14）
   ↳ 计算平均真实波幅的周期

🛑 止损距离：1.0-3.0（默认2.0）
   ↳ 止损 = 进价 ± ATR × 此值
   >>> 更高 = 更宽松，被止损可能性小

⏱️ 最大持有天数：1-20（默认5）
   ↳ 超过此天数未平仓则强制平仓
```

**尝试调整这些参数，看看效果如何改变！**

### 第五步：运行回测

点击 **▶️ 运行回测** 按钮

等待 10-30 秒...

### 第六步：查看最佳策略记忆（新功能！）

回测完成后，你会看到：

```
┌─────────────────────────────────┐
│ 💾 最佳策略                     │
│ 分歧交易策略（改进版）           │
│ 年化收益: 25.43% | 更新于: 2026-03-18  │
│ [📌 加载此配置]                 │
└─────────────────────────────────┘
```

这表示系统自动记住了这个最好的配置！

## ⚙️ 参数调整建议

### 不同市场环境的参数配置

#### 🔵 强势上升趋势
适合频繁交易，降低过滤条件：
```
趋势均线周期：15-20（缩短识别反弹）
波幅扩大倍数：1.1-1.2（放宽波幅要求）
成交量倍数：1.0-1.1（放宽成交量要求）
最大持有天数：7-10（更长期看机会）
```

#### 🟡 震荡盘整市场
适合精选信号，提高过滤条件：
```
趋势均线周期：20-30（确保趋势明确）
波幅扩大倍数：1.4-1.6（严格波幅要求）
成交量倍数：1.3-1.5（严格成交量要求）
最大持有天数：2-3（快速止损）
```

#### 🔴 下降趋势
适合保守交易，不建议做多：
```
趋势均线周期：20（标准设置）
波幅扩大倍数：1.3（标准要求）
成交量倍数：1.2（标准要求）
止损距离：1.5-2.0（缩紧止损）
最大持有天数：2-3（快速退出）
```

## 🆘 常见问题

### Q1：为什么没有看到参数滑块？

**A**：选错了策略！
- 均线交叉策略 → 2个参数（ma_short, ma_long）
- 分歧交易策略（改进版）→ **6个参数** ✓
- 其他策略 → 暂不支持

### Q2：记忆是否永久保存？

**A**：是的！
保存位置：`Data_Hub/storage/.strategy_memory.json`
关闭 GUI 重新打开，记忆仍然存在

### Q3：搜索标的失败怎么办？

**A**：可能的原因：
1. 标的代码错误（如：APPL 应为 AAPL）
2. 网络连接问题
3. Yfinance 数据源暂时不可用

→ 检查代码后重试

### Q4：能否删除某个标的的记忆？

**A**：可以手动编辑 JSON 文件：
```
Data_Hub/storage/.strategy_memory.json

删除相应标的的条目，保存即可
```

### Q5：参数调整会立即生效吗？

**A**：不会。参数调整后需要点击 **▶️ 运行回测** 才会生效

### Q6：一次最多回测多少个标的？

**A**：无限制！记忆系统会为每个标的单独记录最佳配置
