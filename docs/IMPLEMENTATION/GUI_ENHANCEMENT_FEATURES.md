### 2️⃣ 记忆机制（智能记忆最好的策略）

#### 功能说明

**工作原理**：
每次回测完成后，系统自动对比年化收益率

```
回测完成
    ↓
获取年化收益率
    ↓
与记忆中的最好成绩对比
    ↓
如果更高 → 更新记忆 ✅
如果更低 → 保持不变 🔒
```

#### 保存内容

记忆系统保存以下信息：
- 最好的 **策略名称**
- 最好的 **策略参数组合**
- **年化收益率** 百分比
- **更新时间戳**

#### 记忆文件位置

```
Data_Hub/storage/.strategy_memory.json
```

格式示例：
```json
{
  "AAPL": {
    "strategy": "分歧交易策略（改进版）",
    "params": {
      "trend_ma": 20,
      "amplitude_ratio": 1.3,
      "volume_ratio": 1.2,
      "atr_period": 14,
      "stop_loss_atr": 2.0,
      "hold_days": 5
    },
    "annual_return": 0.2543,
    "updated_at": "2026-03-18T14:32:00"
  },
  "TSLA": {
    "strategy": "均线交叉策略",
    "params": {"ma_short": 15, "ma_long": 65},
    "annual_return": 0.1876,
    "updated_at": "2026-03-15T09:15:00"
  }
}
```

#### 在 GUI 中的表现

在回测结果显示区顶部，你会看到：
```
💾 最佳策略 | 年化收益: 25.43% | 更新于: 2026-03-18
[📌 加载此配置]
```

点击 "📌 加载此配置" 按钮可以快速恢复该配置（功能预留）

#### 重点特性

✅ **被动更新**：无需手动保存，自动对比  
✅ **一次一个赢家**：每个标的只记忆最好的配置  
✅ **永久存储**：保存到 JSON 文件，下次打开 GUI 仍可查看  
✅ **前向增长**：只有更好的成绩才会更新  
✅ **多标的独立**：AAPL 的最好配置与 TSLA 无影响

### 3️⃣ 标的搜索与数据下载

#### 功能说明

**工作逻辑**：
1. 点击 "🔍 搜索新标的" 按钮
2. 输入标的代码（如：AAPL, TSLA, BTC-USD）
3. 系统自动检验该标的是否存在
4. 如果存在 → 下载 10+ 年历史数据并永久保存
5. 如果不存在 → 提示错误，不做任何操作

#### 使用步骤

**步骤1：打开搜索面板**
```
侧边栏顶部 [🔍 搜索新标的] 按钮 → 展开搜索框
```

**步骤2：输入标的代码**
```
输入框中输入大写代码（自动转换）
例：aapl → AAPL，tsla → TSLA

支持的格式：
- 美股：AAPL, TSLA, MSFT, GOOGL 等
- 加密货币：BTC-USD, ETH-USD 等
- 指数：^IXIC (Nasdaq), ^GSPC (S&P 500) 等
```

**步骤3：下载并保存**
```
点击 [📥 下载并保存] 按钮
系统会：
1. 连接 yfinance 验证标的存在性
2. 下载 2015-01-01 至今 的全部历史数据
3. 标准化数据格式（open, high, low, close, volume）
4. 保存为 Parquet 文件（高压缩、快读取）
5. 同步显示在"选择标的"列表中
```

#### 数据存储位置

```
Data_Hub/storage/{SYMBOL}.parquet
```

例如下载 AAPL 后会生成：
```
Data_Hub/storage/AAPL.parquet
```

#### 错误处理

**场景1：标的不存在**
```
❌ 未找到标的：XXXX，请检查代码是否正确
```
→ 不会做任何操作，可重试输入

**场景2：网络错误**
```
❌ 下载失败：[具体错误信息]
```
→ 检查网络连接或稍后重试

**场景3：下载成功**
```
✅ AAPL 数据下载成功！
📊 已获取 2756 条数据

[GUI 自动刷新，AAPL 出现在标的列表中]
```

#### 支持的标的类型

| 类型 | 例子 | 说明 |
|------|------|------|
| 美股 | AAPL, TSLA | 直接输入代码 |
| 加密货币 | BTC-USD, ETH-USD | 需要 -USD 后缀 |
| 指数 | ^IXIC, ^GSPC | 需要 ^ 前缀 |
| 外汇 | EURUSD=X | 外汇对格式 |

#### 重点特性

✅ **自动验证**：检查标的是否真实存在  
✅ **长期数据**：下载 10+ 年的历史数据  
✅ **永久存储**：数据保存本地，无需重复下载  
✅ **实时更新**：标的列表自动刷新  
✅ **容错机制**：不存在的标的不做任何操作

## 🛠️ 技术实现细节

### 记忆系统实现

```python
# 核心函数
def update_best_strategy(symbol, strategy_name, params, annual_return):
    """
    只有当年化收益率更高时才更新记忆
    """
    memory = load_memory()
    if symbol not in memory or annual_return > memory[symbol]['annual_return']:
        memory[symbol] = {
            'strategy': strategy_name,
            'params': params,
            'annual_return': annual_return,
            'updated_at': datetime.now().isoformat()
        }
    save_memory(memory)
```

### 数据下载实现

```python
# 新标的下载流程
ticker = yf.Ticker(search_symbol)
```
