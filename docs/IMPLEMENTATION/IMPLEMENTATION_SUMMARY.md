## 🔧 技术实现细节

### 平方根市场冲击模型
```
数学形式：impact = λ × √(Q/V)

其中：
  λ = 市场冲击系数（Impact Coefficient）
  Q = 策略下单量
  V = 当日市场总成交量
  
特点：
  ✓ 对数增长（符合实证观察）
  ✓ 参数可调（适应不同市场）
  ✓ 交易规模安全检查（防止超限）
```

### 双边佣金计算
```
买入成交价 = 原定价格 × (1 + 冲击幅度) × (1 + 佣金率)
卖出成交价 = 原定价格 × (1 - 冲击幅度) × (1 - 佣金率)
```

### 集成架构
```
GUI_Client/app_v2.py
  ├─ 侧边栏配置参数
  ├─ 条件判断：高级模式？
  ├─ 是 ─→ BacktestEngine → AdvancedExchangeSimulator → 高级结果
  └─ 否 ─→ BacktestEngine → 简单结果
     
两种结果同时存储，支持实时对比
```

```python
from Engine_Matrix.advanced_simulator import AdvancedExchangeSimulator, ExchangeConfig
from Strategy_Pool.strategies import STRATEGIES

# 准备数据
signal_data = STRATEGIES[2].backtest(data.copy(), {})

# 配置参数
config = ExchangeConfig(
    commission_rate=0.0005,
    impact_coefficient=0.5,
    max_order_ratio=0.05

result = simulator.run(signal_data, initial_capital=30000)

print(f"Final equity: ${result['equity'].iloc[-1]:,.2f}")
```
