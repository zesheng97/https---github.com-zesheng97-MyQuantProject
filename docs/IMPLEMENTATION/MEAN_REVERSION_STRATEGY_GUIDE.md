# 🆕 均值回归与波动率收割策略 (Mean Reversion Volatility Strategy)

**添加日期**: 2026-03-22  
**策略名称**: 均值回归波动率策略 (Mean Reversion Volatility Harvesting)  
**适用标的**: 高波动、宽幅震荡的股票 (e.g., KRUS)  
**状态**: ✅ **已实装，可立即使用**

---

## 📌 快速概述

这是一个**极度针对高波动标的设计**的量化策略，基于 Gemini 对 KRUS 股票的深度分析。适合那些：
- 呈现极度高波动性 (High Volatility)
- 宽幅震荡 (Wide Range-Bound)
- 趋势跟踪策略容易止损的标的

**核心思想**: 不追高突破，而是**抄底反弹**，通过统计学上的 Z-Score 捕捉极端偏离。

---

## 🎯 策略核心逻辑

### 1️⃣ **买入信号 (Long Entry)**

同时满足两个条件：

```
条件A: Z-Score <= -2.5 (价格极其罕见地下跌)
条件B: 当日成交量 >= 2倍20日均量 (放量见底)
```

**数学表达**:
$$Z = \frac{Price - \mu}{\sigma} \le -2.5$$
$$Volume_{today} \ge 2.0 \times MA_{volume,20d}$$

**含义**: 恐慌性超跌 + 放量确认 = 买入机会

### 2️⃣ **止盈机制 (Exit Strategy)**

**分阶段止盈** (非固定价位):

| 阶段 | 条件 | 操作 | 说明 |
|------|------|------|------|
| **第一阶段** | 价格 >= 均值 μ | 平仓 50% | 锁定部分利润 |
| **第二阶段** | Z-Score >= +1.5 | 平仓剩余 100% | 到达上方阻力 |

### 3️⃣ **止损机制 (Stop Loss)**

**硬性止损** (无条件执行):

```
止损线: Entry Price × (1 + stop_loss_pct)
默认: -4% 止损 (跌破进价4%)

如果触及则无条件平仓
```

**为什么不用时间止损?** KRUS 这类股票容易出现大幅跳空缺口，动态止损容易失效。

## 🛠️ 参数说明

在 Streamlit GUI 中，策略参数分为两类：

### A. Z-Score 与成交量参数

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| **Z-Score 周期** | 60 | 30-120 | 计算均值和标差的回溯周期 |
| **买入阈值** | -2.5 | -4.0 ~ -1.5 | 越负越严格，触发次数越少 |
| **成交量倍数** | 2.0 | 1.0-5.0 | 2.0 = 需要是20日均量的2倍 |
| **成交量均期** | 20 | 10-50 | 计算成交量均线的周期 |

### B. 止盈与止损参数

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| **上方止盈阈值** | +1.5 | 0.5-3.0 | Z-Score 高于此值时卖出所有 |
| **均值止盈比例** | 50% | 10%-100% | 在均值处平仓的比例 |
| **止损百分比** | -4% | -10% ~ -1% | 硬性止损 (负数) |

### 参数理解示例

**保守型** (追求高胜率，牺牲收益):
- Z-Score 周期: 80
- 买入阈值: -3.0 (更严格)
- 成交量倍数: 3.0 (更严格)
- 止盈阈值: +1.0 (更早平仓)

**激进型** (追求高收益，容忍回撤):
- Z-Score 周期: 40
- 买入阈值: -2.0 (更宽松)
- 成交量倍数: 1.5 (更宽松)
- 止盈阈值: +2.0 (更晚平仓)

## 💡 KRUS 案例分析

基于 Gemini 的分析，KRUS 这只股票有以下特征：

1. **历史高点**: $120 (2023年中)
2. **支撑带**: $40-$50 (机构反复验证的流动性池)
3. **阻力线下移**: $120 → $110 → $95 → $80 (多头动能衰减)
4. **当前价格** (2026年初): ~$60-$70 (脱离底部，但距离阻力有空间)

### 该策略在 KRUS 上的表现预期

✅ **优势**:
- 抄底反弹，损失有限 ($40当多，止损$38)
- 成交量放大是强有力的确认信号
- 不会被下降通道的小反弹误导

❌ **劣势**:
- 财报前可能收益被一次性冲击抹除
- 极端行情可能直接跳空止损

## ⚠️ 风险提示

### 策略局限

1. **无法预测财报冲击**: 财报前 3 天可能跳空直接触发止损
2. **收益波动较大**: 赢得少，赔得多的人仓操作
3. **需要精选标的**: 仅适用于高波动股票，不适合蓝筹股

### 使用建议

- ✅ 用于中短期操作 (1-5Y 回溯)
- ✅ 用于高波动标的筛选
- ❌ 不用于短线日推广 (可能信号太稀疏)
- ❌ 不用于企业债/固收资产

## 🔧 故障排查

### 问题1: 没有获得任何买入信号

**原因**: Z-Score 阈值太严格，或标的波动不足

**解决**:
1. 降低 Z-Score 周期 (60 → 40)
2. 放宽买入阈值 (-2.5 → -2.0)
3. 降低成交量倍数 (2.0 → 1.5)

### 问题2: 止损频繁被触发

**原因**: 止损百分比太紧，或标的震荡较大

**解决**:
1. 扩大止损范围 (-4% → -6%)
2. 提高成交量倍数要求 (2.0 → 2.5)
3. 提高 Z-Score 周期 (60 → 80)

### 问题3: 回报率过低

**原因**: Z-Score 阈值过高，或止盈点太早

**解决**:
1. 放宽买入阈值 (-2.5 → -2.0)
2. 延迟止盈点 (+1.5 → +2.0)
3. 关闭 50% 中间止盈 (设为 100%)
