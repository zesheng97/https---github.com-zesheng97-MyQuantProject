## 🎯 策略核心特征

### 1️⃣ 信号生成
```
买入条件 (Long Entry):
1. Z-Score <= -2.5  (极其罕见的下跌)
2. 成交量 >= 2.0倍 20日均量 (放量见底)

两条件同时满足 → 生成买入信号 (signal=1)
```

### 2️⃣ 止盈机制
```
阶段 1 (部分止盈):
  价格 >= SMA均值 → 平仓 50% (signal=0.5)

阶段 2 (完全止盈):
  Z-Score >= 1.5 → 平仓剩余 100% (signal=-1)
```

### 3️⃣ 止损机制
```
硬性止损:
  如果跌破进价 4% (-4%) → 无条件平仓 (signal=-1)
  
  原因: 防止跳空缺口直接穿过浮动止损
```

### 4️⃣ 参数特点
```
默认参数针对 KRUS 这类高波动股票优化：
- 保守的 Z-Score 阈值 (-2.5) 确保质量
- 2.0倍成交量确认避免虚假突破
- 60 日周期提供稳定基准
- -4% 硬性止损防止极端情况
```

## ⚠️ 注意事项

### 1️⃣ 适用范围
```
✅ 适合:
- 高波动股票 (Beta > 1.5)
- 宽幅震荡的标的
- 有明显支撑/阻力的股票
- 中短期操作 (1-5 年回测)

❌ 不适合:
- 蓝筹股 (波动不足)
- 强上升趋势中的股票 (容易止损)
- 流动性很差的股票
- 经常跳空的股票 (可能直接止损)
```

### 2️⃣ 参数调整原则
```
信号太少 → 放宽阈值:
  - 降低 Z-Score 周期 (60 → 40)
  - 提高买入阈值 (-2.5 → -2.0)
  - 降低成交量倍数 (2.0 → 1.5)

止损太频繁 → 放宽止损:
  - 增加止损范围 (-4% → -6%)
  - 提高成交量倍数 (2.0 → 2.5)
  - 增加 Z-Score 周期 (60 → 80)

收益过低 → 提高收益:
  - 放宽买入阈值 (-2.5 → -2.0)
  - 延迟止盈点 (+1.5 → +2.0)
  - 关闭中间 50% 止盈
```

### 3️⃣ 风险管理
```
建议:
- 单笔头寸不超过账户的 2-3%
- 同时监测多个信号，分散风险
- 财报前 3 天考虑禁用买入信号
- 定期回测调整参数
```
