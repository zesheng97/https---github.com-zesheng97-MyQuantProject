#### 2️⃣ **周期性检测模块**
**模块**: `Analytics/reporters/periodicity_analyzer.py`

**三层分析架构**:

```
周期性检测
├── 1. FFT 分析 (快速傅里叶变换)
│   └── 识别主周期，计算周期能量
├── 2. 小波分析 (Continuous Wavelet Transform)
│   └── 捕捉时间-频率变化，识别周期演化
└── 3. KPSS 统计测试 (自相关分析)
    └── 衡量周期性的统计显著性
    
综合评分 = 0.4 × FFT + 0.4 × 小波 + 0.2 × KPSS
```

**关键参数**:
- **周期性阈值**: 0.70 (范围0-1，>0.70视为强周期性)
- **output**: 4个关键指标(FFT分数、小波分数、KPSS分数、综合评分)

**使用示例**:

```python
from Analytics.reporters.periodicity_analyzer import PeriodicityAnalyzer
import pandas as pd

# 1. 初始化分析器
analyzer = PeriodicityAnalyzer(threshold=0.7)

# 2. 分析单支股票
result = analyzer.analyze(df['close'], name='AAPL')

# 3. 查看结果
if result['is_periodic']:
    print(f"✅ {result['symbol']}强周期性")
    print(f"   综合评分: {result['periodicity_score']}")
    print(f"   主周期(FFT): {result['dominant_period_fft']} 天")
else:
    print(f"❌ {result['symbol']}周期性较弱")
    print(f"   建议使用: 趋势跟踪 / 均值回归策略")
```

**结果解读**:

| 综合评分 | 周期性特征 | 推荐策略 |
|---------|----------|--------|
| **>0.80** | 强周期 | ⭐ 周期性趋势交易、周期相位对齐 |
| **0.70-0.80** | 明显周期 | ⭐ 周期性均值回归、周期交易 |
| **0.50-0.70** | 较弱周期 | △ 可配合其他策略 |
| **<0.50** | 弱周期/趋势 | ✗ 建议用趋势或均值回归策略 |

#### 3️⃣ **周期性交易策略**
**位置**: `Strategy_Pool/custom/cyclical_strategies.py`

**三大策略**:

##### A. 周期性趋势交易策略 (CyclicalTrendStrategy)
```
逻辑: 在周期低点买入 → 周期高点卖出 (趋势跟踪)

参数:
  min_period (int): 最短周期，默认 5 天
  max_period (int): 最长周期，默认 60 天
  signal_strength (float): 信号强度阈值，0.3-1.0
  position_size (float): 头寸规模，0.1-1.0
  atr_stop_loss (float): ATR止损倍数，默认 2.0

适用场景: 具有明显周期性的股票
```

##### B. 周期性均值回归策略 (CyclicalMeanReversionStrategy)
```
逻辑: 利用周期内的超买超卖 → 均值回归交易

参数:
  period (int): 周期长度，10-90 天
  zscore_threshold (float): Z-score超买超卖阈值
  lookback (int): 回看窗口，20-120 天

适用场景: 周期振荡明显的股票
```

##### C. 周期相位对齐策略 (CyclicalPhaseAlignmentStrategy)
```
逻辑: 基于周期相位的精准交易
      在周期上升沿买入，下降沿卖出

参数:
  period (int): 主周期
  phase_buy_start (float): 买入相位起点 (0-1)
  phase_buy_end (float): 买入相位终点 (0-1)
  min_reversion (float): 最小回归系数

适用场景: 周期规律明显、相位稳定的股票
```

#### 4️⃣ **Sharpe比评级与可视化**
**模块**: `Analytics/reporters/sharpe_rating.py`

**5级评级系统**:

```
       绿色 ▓▓▓▓▓        黄色 ▓▓▓▓▓        橙色 ▓▓▓▓▓        红色 ▓▓▓▓▓
       
Sharpe  极好(2.0+)    很好(1.5-2.0)   良好(1.0-1.5)   一般(0.5-1.0)   差(<0.5)
Ratio   百分位95-100   百分位75-95     百分位50-75     百分位25-50     百分位0-25
```

**核心特性**:

1. **彩虹仪表盘图** (Rainbow Gauge Chart)
   - Plotly交互式图表
   - 实时显示Sharpe Ratio等级
   - 5色彩虹背景，直观评估

2. **详细评级卡片** (Rating Card)
   - 中英文评级标签
   - 百分位排名
   - 相关指标关联显示
   - 优劣分析描述

3. **策略对比** (Strategy Comparison)
   - 多策略Sharpe对比柱状图
   - 排名看板
   - 参考线标注（2.0/1.5/1.0/0.5阈值）

**使用示例**:

```python

st.plotly_chart(fig)  # Streamlit显示

    '布林带': 1.2,
```

## 🔧 技术细节

### 周期性检测的数学原理

```
FFT分析流程:
1. 输入: 价格序列
2. 去趋势: detrended = price - polyfit(2阶多项式)
3. FFT变换: X = FFT(detrended)
4. 频率分析: freq = np.argmax(|X|)
5. 主周期: period = n / freq
6. 周期强度: 主峰能量 / 平均能量

小波分析:
1. 使用Morlet小波
2. CWT在多个尺度执行
3. 识别时间的频率变化
4. 捕捉周期演化特征

综合评分 = 加权平均
```

### Sharpe Ratio计算

```
Sharpe = (策略年化收益 - 无风险率) / 策略收益波动率
       ≈ (策略年化收益) / 策略收益波动率

5级标准:
├─ 极好: ≥ 2.0    (专业基金经理水准)
├─ 很好: 1.5-2.0  (优秀)
├─ 良好: 1.0-1.5  (可行)
├─ 一般: 0.5-1.0  (需改进)
└─ 差:   < 0.5    (不推荐)
```

## ⚠️ 注意事项

1. **IPO日期精度**: 某些股票的IPO日期可能不准确，脚本有备用方案
2. **数据成本**: 下载700+股票耗时较长，建议在夜间运行
3. **周期性阈值**: 0.7为推荐值，可根据需求调整 (0.6更保守, 0.8更严格)
4. **Sharpe Ratio局限**: 假设收益率正态分布，极端行情可能失效
5. **回测局限**: 不考虑交易成本、滑点、流动性等实际因素
